Inline JavaScript is when you write JavaScript code directly on an HTML page in a script tag. This is the simpiliest way to get JavaScript on a page but should only be used for learning purposes. The problems with inline JavaScript include:

  • No ability to reuse JavaScript code
  • Variables and functions are added to global namespace
  • Not easy to add libraries or othe code that you have written. Have to add code to page and make sure that everything is in the right order.
